# Moid Addon 🌑

A specialized utility expansion for the **Meteor Client** (1.21+), designed with a focus on performance, internationalization, and "vibe-coded" logic. 

> **Compatibility:** Built for Fabric 1.21+.

---

## 🛠️ Installation

1. **Prerequisites:** Ensure you have [Meteor Client](https://meteorclient.com/) installed for Fabric 1.21.x.
2. **Download:** Go to the [Releases](https://github.com/DioM34/Moid-Addon/releases) tab and download the latest `.jar` file.
3. **Move:** Drop the `.jar` into your Minecraft `mods` folder.
4. **Launch:** Start Minecraft and enable **Moid Addon** in the Meteor modules menu.

## 🧩 The Full Suite

### ⚔️ Combat & Movement
| Module | Description |
| :--- | :--- |
| **Moid-Backtrack** | Freezes entity positions by delaying inbound packets. Includes a **Ghost Renderer** to show exactly where to hit. |
| **Moid-KillAura** | Advanced combat automation with customizable targeting and rotation logic. |
| **Moid-TriggerBot** | Precision auto-attack when your crosshair meets an enemy. |
| **Moid-Fly** | Optimized flight logic for bypassing movement checks on various servers. |
| **Moid-Velocity** | Take zero or reduced knockback. Essential for maintaining ground in 1v1s. |
| **Moid-Reach** | Extends your interaction range, letting you get the first hit every time. |

### 👁️ Visuals & ESP
| Module | Description |
| :--- | :--- |
| **Moid-HUD** | The command center. Shows FPS, Ping, Coords, and Active Modules in a clean, high-performance overlay. |
| **Moid-HitESP** | Renders a color-fading box exactly where your hits connect. |
| **Moid-GhostESP** | Visualizes the "ghost" positions of players during backtrack or lag spikes. |
| **Moid-CircleESP** | Clean, geometric highlights around entities for better spatial awareness. |
| **Moid-Trails** | Leaves a customizable trail behind you or other players to track movement paths. |
| **Moid-Trajectory** | Predicts the landing spot of pearls, arrows, and potions with high accuracy. |

### ⚙️ Utility & Misc
| Module | Description |
| :--- | :--- |
| **Moid-PingSpoof** | Masks your true connection. Features **Static** and **Randomize** modes to mimic real-world network jitter. |
| **Moid-TargetFocus** | Locks your camera or priority on a specific entity to ensure they never escape. |
| **Moid-Fakelag** | Intentionally disrupts your own packet flow to confuse opponents (FakeLag). |
| **Moid-AutoClicker** | A highly customizable click-macro for non-aura combat styles. |
